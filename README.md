# My Windhawk Mods

My personal collection of Windhawk mods.

> [!Note]
> All mods in this repository are primarily **Proof of Concept (PoC)** and currently in **Beta**. They are highly experimental, potentially unstable, and might cause unexpected system issues. Use them at your own risk.

This repository is used for development. Stable mods may eventually be submitted to [The official collection of Windhawk mods](https://github.com/ramensoftware/windhawk-mods).

## Mods

### Edge Window Docker
Docks active window to nearest screen edges, auto-hide with slide animation.
